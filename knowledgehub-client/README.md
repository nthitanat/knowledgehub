# Knowledge Hub - Chulalongkorn University

Knowledge and Innovation Hub for Glocal Marketing

## Overview

Platform for community products, storytelling, co-design, and mini-courses connecting Thai communities with global markets (especially GCC/Arab markets).

## Features (Phase 1)

- ✅ Bilingual support (TH/EN)
- ✅ Community profiles with storytelling
- ✅ Digital Showroom
- ✅ Localization notes for products
- ✅ Role-based access
- ✅ Material Design 3 with CU branding

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Project Structure

```
src/
├── api/                    # API service layer (JSON mock data)
├── components/             # Reusable components
│   ├── common/            # Shared components
│   └── [domain]/          # Domain-specific components
├── pages/                 # Route-level pages
├── context/               # React Context providers
├── styles/                # Global styles (SCSS)
├── data/                  # Mock JSON data
└── App.js                # Main app with routing
```
